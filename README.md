# santana

Webmap simples (Leaflet) com 3 planos de fundo e pesquisa por `zona_eleit`.

## Publicação (GitHub Pages)

Depois que o GitHub Actions terminar o deploy, o site fica em:

- `https://oaluap.github.io/santana/`

## Rodar localmente (opcional)

```bash
node server.js
```

Abra `http://127.0.0.1:8000/`.
